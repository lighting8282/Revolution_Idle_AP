# Revolution Idle Setup Guide

> **Status: early scaffold (v0.1).** Logic and the game-side mod are still in development.

## Required Software

- [Archipelago](https://github.com/ArchipelagoMW/Archipelago/releases)
- Revolution Idle (Steam)
- The Revolution Idle BepInEx mod (not yet released)

## Generating a Game

1. Configure your YAML options (Goal, Trap Chance, Death Link).
2. Generate as normal, either locally or on the website.

## Connecting

The BepInEx mod connects the running game to the Archipelago server. Setup instructions will be
added here once the mod is available.
